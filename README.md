# 🛡 Campus Guardian – README

## Overview

Campus Guardian is a web-based safety application designed to help students quickly send alerts during emergencies. The system provides options for emergency help, health assistance, and complaint reporting while capturing the user's location for faster response.

## Features

* 🚨 Emergency panic button
* 🩺 Health assistance request
* ⚠ Complaint / lost item reporting
* 📍 Location tracking using Geolocation API
* 📱 Responsive and simple user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download the project files.
2. Open the **index.html** file in any web browser.
3. Allow location access when prompted.
4. Use the buttons to send alerts.
